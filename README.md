# food-web 🍔

A modern AI-powered food ordering web application built with HTML, CSS, and Vanilla JavaScript.

## Features

- 🤖 AI-powered food recommendations chatbot
- 🍕 Real food images from Unsplash CDN
- 🛒 Shopping cart with quantity management
- 📦 Order tracking with live status
- 👤 User dashboard with order history
- 🌙 Dark/Light mode toggle
- 📱 Fully responsive mobile design
- 🎨 Premium UI with Playfair Display + Plus Jakarta Sans fonts

## Tech Stack

- HTML5 + Vanilla CSS + Vanilla JavaScript
- Google Fonts: Playfair Display, Plus Jakarta Sans, Space Grotesk
- Unsplash CDN for food photography

## Live Preview

Open `foodiehub.html` in any browser to view the app.
